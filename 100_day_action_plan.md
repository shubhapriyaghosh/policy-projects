# 100-Day Action Plan

## Days 0-30: Diagnose
- Map service journey.
- Identify data sources.
- Define SLA and escalation rules.
- Establish baseline metrics.

## Days 31-60: Build
- Build control-tower dashboard.
- Prototype AI triage model.
- Create escalation workflow.
- Set citizen communication templates.

## Days 61-90: Pilot
- Pilot in high-volume departments.
- Monitor accuracy, SLA improvement and citizen feedback.
- Conduct bias and error review.

## Days 91-100: Scale decision
- Review pilot metrics.
- Finalise governance SOP.
- Approve scale roadmap.
