# Problem Tree

## Core problem
Delayed and opaque public-service delivery.

## Direct causes
- Manual verification
- Fragmented data
- Weak SLA tracking
- No early-warning system
- Limited citizen communication

## Root causes
- Departmental silos
- Poor workflow ownership
- Lack of digital integration
- Limited performance accountability

## Effects
- Citizen dissatisfaction
- Repeated grievances
- Higher administrative burden
- Lower trust in governance
