# Dashboard Design Note

## Dashboard layers
1. Executive summary: SLA, TAT, backlog, citizen score.
2. Department drill-down: applications, grievances, reopen rates.
3. Exception management: delayed cases, high-risk departments.
4. Citizen experience: feedback score, complaint themes.

## Core KPIs
- SLA met percentage
- Average turnaround time
- Backlog aging
- Grievance rate
- Reopen rate
- Citizen satisfaction score

## Review cadence
- Daily: delayed cases and critical backlogs
- Weekly: department-level review
- Monthly: policy and process redesign decisions
